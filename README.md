# 💰 Apresentação do Projeto: App Money (Gestão Financeira)

Bem-vindo ao nosso projeto prático de Gerenciamento Financeiro! Nesta etapa inicial, vamos conhecer a proposta do aplicativo **Money**, entender suas principais funcionalidades e aceitar um grande desafio prático.

## 🎯 O Desafio: Mão na Massa!
**Pare, pense e codifique!** A grande provocação desta aula é que você **pause o curso agora mesmo** e tente implementar todo o aplicativo por conta própria antes de ver a resolução. Você já possui a base de conhecimento necessária para desenvolver a maior parte das funcionalidades apresentadas.

Tente chegar o mais longe possível e consulte as documentações oficiais sempre que precisar. Nas próximas aulas (e nos próximos commits deste repositório), construiremos o app passo a passo, mostrando uma das possíveis resoluções, introduzindo conceitos de formulários e aplicando boas práticas de desenvolvimento.

## 📱 Funcionalidades do App
O aplicativo será dividido em três abas principais:

1. **➕ Adicionar Transação:** Tela para registrar uma nova movimentação. Você poderá adicionar uma compra (ex: um sanduíche), informando o valor, a data e a categoria correspondente.
2. **📋 Listar Transações:** Uma tela dedicada a exibir o histórico com todas as entradas e saídas que você cadastrou.
3. **📊 Resumo:** Um painel consolidado juntando tudo o que foi ganho (renda) e detalhando as perdas (gastos) divididas por categorias (ex: Alimentação, Casa, Educação e Viagens).
